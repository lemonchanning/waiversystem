Functional characteristics
Wave provides platforms for social networking and donation, and each platform has perfect supporting functions.
Social platform: 
	sign up
	sign in
	Upload Avatar
	Edit resume
	Post
	comment
Donation platform: 
	launching fund-raising projects
	contribution
	comment

Environmental dependence

Support MacOS, windows and other operating systems
Java virtual machine needs to be installed on the platform
Java version 1.6 or above, depending on spring 3
Tomcat 6.0 or later
The database version is MySQL 8.0 and Hibernate version is 3.4 or higher
Windows 7 or later is recommended

Deployment steps

1. Import database
2. Import project
3. Create server
4. Deploy project to server
